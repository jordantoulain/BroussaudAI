# Local Chatbot

Application de chat locale avec interface web pour interagir avec un modèle de langage.

## Fonctionnalités

- **Chat en temps réel** avec un assistant IA local
- **Gestion des conversations** : historique, création, suppression
- **Mode Administration** pour configurer et gérer le système
- **Authentification** et gestion des profils utilisateurs

## Prérequis

- Node.js 18+
- Un backend compatible (voir [api/](./api/))

## Installation

```bash
cd web
npm install
npm run dev
```

L'application sera disponible sur `http://localhost:3000`.

## Structure

- `api/` – Backend
- `web/` – Frontend Next.js
- `docs/` – Documentation technique
